# 扩展案例

这个是一个扩展的demo示例，用以帮助开发者学习和使用嘉立创EDA API，覆盖了弹窗、按钮、图元增删改查等接口，其功能仅做扩展开发的学习案例。其包含了图元读取控制、弹窗、吐司消息、日志输出等关键例子。

提示：这个扩展案例的功能不适合正常使用，仅做开发参考。
