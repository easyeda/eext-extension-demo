#拡張ケース

これは開発者が嘉立創EDA APIを学習し、使用するのを支援するための拡張demoの例であり、ポップアップ、ボタン、エンティティの追加削除変更などのインタフェースをカバーし、その機能は概略学習のみを行う。

拡張ケース//このケースは原理図とPCBの両方に有効です
｜―――ビット番号をランダムに再割り当て
｜――――部品属性を一括追加する...//後弾窓をクリックし、属性名、属性値を入力し、確定してすべての構成部品に適用
｜――――テキストを一括配置する...//クリック後にポップアップ、テキスト、数、間隔、キャンバスに自動的に配置、デフォルトフォントを使用
｜――――エンティティを削除する...//後弾窓をクリックし、エンティティをチェックし、確認後に同型エンティティをすべて削除します。キャンバスエンティティ：`コンポーネント、ワイヤ、テキスト、ポート`（原理図）、`素子、ワイヤ、パッド、ビア、銅張り、充填、線、テキスト`（PCB）
————エンティティ統計...//クリックするとポップアップ窓が表示され、キャンバスエンティティと対応する数が表示されます。キャンバスエンティティ：`コンポーネント、ワイヤ、テキスト、ポート`（原理図）、`素子、ワイヤ、パッド、ビア、銅張り、充填、線、テキスト`（PCB）
